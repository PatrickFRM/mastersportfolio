Jaccard Coefficient Calculation

The table shows the pathological test results for three individuals.

Name Gender Fever Cough Test-1 Test-2 Test-3 Test-4
Jack M Y N P N N A
Mary F Y N P A P N
Jim M Y P N N N A

Calculate Jaccard coefficient for the following pairs:

  • (Jack, Mary)
  • (Jack, Jim)
  • (Jim, Mary)

First of all we covert these into binary but first we assume the values such as A and N in Test-4 then proceed to assign them 1 and 0s creating the below table:

Name Gender Fever Cough Test-1 Test-2 Test-3 Test-4
Jack M 1 0 1 0 0 0
Mary F 1 0 1 0 1 0
Jim M 1 1 0 0 0 0

We are now ready to find out the Jaccard coefficient.

Jack and Mary Coefficient

Jack and Mary intersect on 2 tests out of the 6 so the calculation of Intersection/Union is 4/6 giving the Jaccard coefficient of 0.33.

Jack and Jim Coefficient

Jack and Jim intersect on 4 tests out of the 6 so the calculation of Intersection/Union is 4/6 giving the Jaccard coefficient of 0.67.

Jim and Mary Coefficient

Jim and Mary intersect on 3 tests out of the 4 so the calculation of Intersection/Union is 2/4 giving the Jaccard coefficient of 0.75.
